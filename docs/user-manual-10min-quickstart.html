教创搭子用户手册（10分钟速读版）

适用对象：第一次上手的老师 · 目标：一节课跑通“课前-课中-课后”
主线： 课前导入学员名单并完成分组/建队/排座； 课中使用白板互动并收缴作业； 课后用词库巩固知识点并查看班级积分。

课前（3-4分钟）

  • 导入学员名单，检查重名和空行。
  • 自动分组后拖拽微调，形成队伍结构。
  • 按教室实际行列、走道、禁用位完成排座。

课中（4-5分钟）

  • 创建白板话题，发布二维码。
  • 收集学生作业卡片并进行点评/置顶。
  • 穿插倒计时、随机点名，保持课堂节奏。

课后（2-3分钟）

  • 将本节知识点整理进词库并生成闪卡。
  • 查看班级积分与个人表现变化。
  • 导出课堂数据并按班级日期归档。

完成标志

  • 名单、分组、座位三项可复用。
  • 白板作业可追踪、可点评。
  • 词库与积分可用于复盘和激励。
新手建议：先跑通主线，不追求一次用全功能。先稳定流程，再扩展玩法。
